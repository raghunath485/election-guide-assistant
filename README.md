# Election Guide Assistant

An interactive civic education assistant for Challenge 2. It helps users understand election timelines, voter preparation steps, ballot planning, and post-election processes in a neutral and easy-to-follow way.

## Features

- Streamlit assistant with topic shortcuts and chat-style answers
- Personalized voting plan builder
- Downloadable action-plan checklist
- Readiness tracker with progress score
- Election timeline explainer
- Short quiz with feedback
- Searchable glossary of election terms
- Static HTML/CSS/JavaScript version included

## Run The Streamlit App

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```

## Run The Static App

Open `index.html` in a browser.

## Important Note

Election rules, dates, ID requirements, and voting options vary by location and can change. This project explains the general process and reminds users to verify exact requirements with their official state or local election office.
